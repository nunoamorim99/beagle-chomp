# App icons

Generate three PNGs from a 1024×1024 master (a beagle face on the `#0b0b16`
background works well) and drop them here:

- `icon-192.png` — 192×192
- `icon-512.png` — 512×512
- `icon-512-maskable.png` — 512×512, artwork inside the safe zone (~80% centre)

Until these exist the app still runs; installability just needs them present.
